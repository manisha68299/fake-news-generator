Fake News Headline Generator

A simple and interactive Flask web application that creates fake news headlines for fun.
The project includes user authentication, a clean UI, and dynamic content generation.

Features

User Signup and Login

Session-based authentication

Random fake news headline generation

Clean and modern UI

Built using Flask

Easy to run and understand
